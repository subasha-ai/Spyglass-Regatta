📐 How Scoring Works

Each building is ranked #1–18 for every metric. Ranks are weighted and summed. Lowest total score wins (like golf).

Score = Σ (Rank × Metric Weight × Category Weight)
🔢 The Process
1. Rank
For each metric, rank all buildings. Best = #1. Ties share ranks.
2. Weight
Multiply rank by metric weight and category weight.
3. Sum
Add all weighted ranks across all 21 metrics.
4. Rank
Lowest total score = #1 overall.

Best possible: 1.00 (rank #1 in everything) · Worst possible: 18.00
