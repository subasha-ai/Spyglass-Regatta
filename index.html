📐 Scoring Formula — Rank-Based System

Each building is ranked #1-17 for every metric. These ranks are then weighted and summed. Lowest total score wins (like golf!).

Total Score = Σ (Metric Rank × Metric Weight × Category Weight)
🔢 How It Works
1
Rank Each Metric
For each metric, rank all 17 buildings based on performance. Best performer = Rank 1. Ties share the same rank (e.g., 5 buildings with 5-star rating all get rank 1). The next building starts at rank 6.
2
Apply Weights
Multiply each rank by: Metric Weight (within category) × Category Weight (overall importance). Example: Rank 6 × 0.20 (QM weight) × 0.26 (Care weight) = 0.312 points
3
Sum All Metrics
Add up all weighted ranks across all 21 metrics in all 7 categories to get the building's total score.
4
Lowest Score Wins
Buildings are ranked by total score — the building with the lowest weighted rank total is #1 (like golf scoring).
📝 Example: QM Rating (Care Category)

QM Rating has a metric weight of 20% within the Care category, which has a category weight of 26%.

⚠️ Missing Data

If a metric is missing for all buildings, that metric is excluded from scoring. If a building has missing data for a metric that others have, it receives the worst rank (17) for that metric to avoid rewarding incomplete data.
